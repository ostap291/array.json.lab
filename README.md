# Лабораторна робота: Обробка масивів та JSON у C++

Цей репозиторій містить виконання лабораторної роботи з програмування (C++). Програма генерує масив випадкових чисел, виконує математичні обчислення, власне сортування та зберігає результати у форматах .txt та .json.

## Основний функціонал
- Генерація та обробка: Створення одновимірного масиву, поділ на n груп та обчислення проміжних сум.
- Сортування (Bubble Sort): Власна реалізація алгоритму сортування «бульбашкою» за зростанням та спаданням.
- Фільтрація: Пошук унікальних елементів масиву та підрахунок кількості дублікатів.
- Файлове введення/виведення: - Проміжні розрахунки (суми груп) зберігаються у звичайний текстовий файл e.txt.
  - Фінальні результати (масиви та суми) серіалізуються та зберігаються у структурований файл results.json.
- Unit-тестування: Логіка сортування покрита автоматичними тестами з використанням макросу assert.

## Структура проєкту
- main.cpp — точка входу для запуску основної програми.
- test_main.cpp — незалежний файл для запуску юніт-тестів (ізольовано від основного `main`).
- ArrayTask.h / ArrayTask.cpp — клас, що інкапсулює всю логіку обробки масивів та роботи з файлами.
- json.hpp — зовнішня бібліотека nlohmann/json для зручної роботи з форматом JSON.

- C++ (Standard Library: <iostream>, <vector>, <fstream>, `<cassert>`)
- Бібліотека [JSON for Modern C++](https://github.com/nlohmann/json)
